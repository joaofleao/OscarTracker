# OscarTracker

### Description
An unofficial movie tracker and social platform for the Academy Awards

### Steps to Run
1. Clone the repository
2. Install and configure Node, Android and iOS enviroment
3. Add the api keys for both TMDB and the Firebase on a .env file on the root of the project
4. Run `npm install` to install all the dependencies
5. Run `npm start` to start Expo, then click `i` for iOS and `a` for Android
